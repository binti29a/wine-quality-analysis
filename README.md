\f0\b\fs26  Project Overview
\f1\b0\fs24 \
This project analyzes a red wine quality dataset to identify how chemical properties such as alcohol, volatile acidity, and sulphates relate to wine quality. The dataset was imported into MySQL Workbench, where SQL queries were used to explore quality distribution and compare key characteristics across low-, medium-, and high-quality wines.\

\f0\b\fs28 Business Questions
\f1\b0\fs24 \
1. How are wines distributed across different quality scores?\
2. How does alcohol content vary across different wine quality levels?\
3. Is there a relationship between volatile acidity and wine quality?\
4. How do sulphate levels differ across wine quality categories?\
5. What characteristics define the highest-quality wines in the dataset?\
6. How do key chemical properties differ between low-, medium-, and high-quality wines?\
\

\f0\b\fs28  Key Insights
\f1\b0\fs24 \
- Most wines fall in the mid-quality range, especially quality scores 5 and 6.\
- Higher-quality wines tend to have higher alcohol content.\
- Volatile acidity decreases as wine quality increases.\
- Sulphate levels generally increase with wine quality.\
- High-quality wines are characterized by higher alcohol, lower volatile acidity, and higher sulphates.\
\

\f0\b\fs28  Conclusion
\f1\b0\fs24 \
This analysis shows that wine quality is strongly associated with several chemical characteristics. Higher-quality wines generally have higher alcohol content, lower volatile acidity, and higher sulphate levels.\
